Install rails => gem install rails
Create new application => rails new "NameOfApplication" -d mysql
Add routing to the file  config/routes.rb => get "nameofcontroller/nameofaction"
Remove stylesheet and javascript tag from application.html.erb inside views/layout/
Create new controller => rails generate controller nameofcontroller nameofview (name of view is optional)


Creating model and running rake commands :-
Run rake db:create for creating databases in phpmyadmin
Run rails g model "NameOfModel" for generating model and db schema file as well
open db/migrate/NameOfFile which is generated by the model and provide column names in the same and afterwards 
Run rake db:migrate for creating the table in back end
